## 🚀 How To Use

### Push to GitHub (3 steps)
```bash
# 1. Initialize Git
cd superbrain
git init
git add .
git commit -m "Initial commit: Production-ready SuperBrain"

# 2. Add remote (create repo on github.com first)
git remote add origin https://github.com/USERNAME/superbrain.git

# 3. Push
git push -u origin main
```

### Deploy Backend
```bash
# Docker (recommended)
cd backend
docker-compose up -d

# Ready in 30 seconds! Check health:
curl http://localhost:5000/health
```

### Test Frontend
```bash
# Setup
cd superbrain-app
npm install

# Start
npx expo start

# Press 'a' for Android
```
